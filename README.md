## アプリケーションの起動 (Localhost)
```bash
$ cd ResponderKeras
$ python run.py
```

## 実装・実行環境 (versions)
* OS ... MacOS Catalina 10.15 beta
* IDE ... PyCharm 2019.1.1
* Browser ... Safari 13.0.1  
* Python and Libraries   
    - Python ... 3.7.4  
    - Responder ... 1.3.2  
      (Jinja2 ... 2.10.1)  
      (Starlette ... 0.12.9)
    - Keras ... 2.2.4  
    - matplotlib ... 3.1.1
    - numpy ... 1.16.4
