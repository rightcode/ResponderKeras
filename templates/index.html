Web上で気軽に機械学習を学ぼう！

まずは以下から学習させたいタスクを選択しましょう。

MNIST Dataset

タスク　: 画像分類
入力　　: 28x28のグレースケール画像
ラベル　: 0~9 (欠損なし)
データ　: 計70000枚
特徴　　: 簡易画像分類ベンチマーク

配布元　: THE MNIST DATABAE

UCI Iris Dataset

タスク　: 単純クラス分類
入力　　: 4次元の実数値
[sepal length, sepal width, petal length, petal width]
ラベル　: 3クラス
[iris-setosa, iris-versicolour, iris-virginica]
データ　: 計150枚
特徴　　: 簡易・軽量多クラス分類ベンチマーク

配布元　: UCI Machine Learning Repository: Iris Data Set

UCI Wine Dataset

タスク　: 単純クラス分類
入力　　: 13次元の実数値
　　　1) Alcohol
　　　2) Malic acid
　　　3) Ash
　　　4) Alcalinity of ash
　　　5) Magnesium
　　　6) Total phenols
　　　7) Flavanoids
　　　8) Nonflavanoid phenols
　　　9) Proanthocyanins
　　　10)Color intensity
　　　11)Hue
　　　12)OD280/OD315 of diluted wines
　　　13)Proline
ラベル　: 3クラス (品種)
データ　: 計178枚
特徴　　: 簡易・軽量多クラス分類ベンチマーク

配布元　: UCI Machine Learning Repository: Wine Data Set
